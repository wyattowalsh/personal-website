Riso is a modular project scaffolding system built on [Copier](https://copier.readthedocs.io/). It generates production-ready Python and Node.js projects with configurable modules for CLI tools, APIs, documentation sites, and full SaaS starters.

## Why Riso?

Starting new projects often means copying boilerplate, configuring linters, setting up CI/CD, and wiring together documentation. Riso automates all of this with a single interactive command:

```bash
copier copy gh:wyattowalsh/riso my-project
```

Answer the prompts and get a fully configured project with:

- Pre-configured linting (Ruff, ty, Pylint, ESLint)
- GitHub Actions workflows with matrix testing
- Docker and deployment patterns
- Documentation site scaffolding
- 90% test coverage requirements enforced

## Quick Start

```bash
# Install Copier
pip install copier

# Create a new project
copier copy gh:wyattowalsh/riso my-project

# Follow the interactive prompts
```

After generation, the project includes a `Makefile` and task runner commands for common operations:

```bash
# Run quality checks
make quality

# Run tests
uv run pytest

# Build documentation
make docs
```
