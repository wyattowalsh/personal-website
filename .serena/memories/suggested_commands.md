Common development commands:
- pnpm dev — run the Next.js dev server on port 3000.
- pnpm preprocess — rebuild the search index/cache.
- pnpm build — production build.
- pnpm lint — run ESLint.
- pnpm typecheck — run TypeScript in strict noEmit mode.
- pnpm test — run Vitest.
- pnpm storybook — run Storybook on port 6006.
- pnpm new-post --title "X" --tags "A,B" — create a new blog post in CLI mode.
System utilities available on Darwin include git, ls, cd, find, rg, bash, curl, and gh.
