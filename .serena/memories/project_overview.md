Personal website built with Next.js 16 App Router, React 19, TypeScript 5, TailwindCSS 4, MDX, and shadcn/ui. The repo contains the site app under app/, shared UI under components/, content/posts/{slug}/index.mdx for blog sources, server/data logic under lib/, and build/preprocess utilities under scripts/. The site includes blog routes, admin routes, API routes, and supporting docs under docs/.
