# docs/ — Fumadocs Documentation Site

> **Separate pnpm workspace** — run commands with `pnpm --filter docs <cmd>`

## Stack

- **Framework:** Fumadocs (core 15.5.3, mdx 11.6.9, ui 15.5.3)
- **Runtime:** Next.js 15, React 19, TypeScript 5
- **Styling:** TailwindCSS 4, fumadocs-ui theme

## Commands

```bash
pnpm --filter docs dev        # Dev server
pnpm --filter docs build      # Production build
```

## Structure

```
docs/
├── source.config.ts          # Fumadocs content source config
├── next.config.mjs           # Next.js config (createMDX wrapper)
├── mdx-components.tsx        # MDX component overrides
├── tsconfig.json             # TypeScript config
├── package.json              # Workspace dependencies
├── app/                      # Next.js App Router (docs UI)
├── components/               # Doc-specific components
├── lib/                      # Doc utilities
├── content/docs/             # Documentation content (MDX)
│   ├── index.mdx             # Docs landing page
│   ├── meta.json             # Navigation/sidebar config
│   ├── architecture/         # System architecture (5 pages)
│   ├── content/              # Content authoring guides (5 pages)
│   ├── features/             # Feature documentation (6 pages)
│   ├── operations/           # Ops & deployment (4 pages)
│   └── reference/            # API & library reference (5 pages)
├── public/                   # Static assets for docs
└── .source/                  # Auto-generated — DO NOT EDIT
```

## Content Conventions

- MDX pages at `content/docs/{category}/{page}.mdx`
- Navigation order controlled by `meta.json`
- Frontmatter: `title`, `description` (used by Fumadocs nav/search)
- Use standard MDX components (code blocks, callouts, tabs)

## Config Files

| File | Purpose |
|------|---------|
| `source.config.ts` | `defineDocs()` — content source definition |
| `next.config.mjs` | `createMDX()` wrapper for MDX processing |
| `meta.json` | Sidebar navigation tree |
| `tsconfig.json` | Path aliases: `@/.source`, `@/*` |

## Boundaries

| | Rule |
|---|------|
| 🚫 **Never** | Edit `.source/` — auto-generated by fumadocs-mdx |
| ✅ **Always** | Keep docs content aligned with main site codebase |
| ✅ **Always** | Run `pnpm --filter docs build` after content changes |
| ⚠️ **Ask** | Adding fumadocs plugins or changing framework config |
